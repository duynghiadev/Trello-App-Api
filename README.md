## Node.js + Expressjs + MongoDB - Building an APIs like Trello from scratch

## Requirements

- **nodejs >= v14.7.0**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone http://git_url_clone <project_dir>
$ cd <project_dir>
$ npm install
$ npm start
```
